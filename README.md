# Estructuras de Datos Lineales: Pilas y Colas

Este proyecto contiene la implementación en Java de dos ejercicios prácticos:
1. **Validación de Paréntesis:** Uso de la clase `Stack` para asegurar la integridad de expresiones matemáticas.
2. **Simulador de Procesos:** Uso de la interfaz `Queue` para gestionar tareas bajo el principio FIFO (First In, First Out).

### Requisitos
* JDK 17 o superior.
* IDE de preferencia (IntelliJ, Eclipse o VS Code).
